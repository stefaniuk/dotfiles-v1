dotfiles
========

There are two ways you can install the `dotfiles`:

    git clone https://github.com/stefaniuk/dotfiles.git
    ./dotfiles/install.sh

or

    wget https://raw.githubusercontent.com/stefaniuk/dotfiles/master/install.sh -O - | /bin/bash
